## 如何推导大O阶？

1. 用常数1取代运行时间中的所有加法常数
2. 如果最高项存在且不是1，去除与这个项目相乘的常数
3. 得到的最后结果就是大O阶

## 线性阶：

```c
int i , n = 100 , sum = 0;
for (i = 0 ; i < n ; i++)
{
	sum += i ; 
}
```

时间复杂度为O(n)，因为循环执行n次

## 平方阶：

```c
# 嵌套循环
int i,j,n = 100;
for ( i = 0; j < n; i++ )
{
	for ( j = 0; j < n; j++)
	{
		printf("hello")
	}
}
```

时间复杂度O(n^2)

## 对数阶：

```
int i = 1, n = 100;
while( i < n)
{
	i = i * 2;
}
```

2^x = n 所以x = log(2)n
